Open a Text Editor
 Nano (Simple Command-Line Editor):nano prometheus_setup.py
 Vim (Advanced Command-Line Editor):vim prometheus_setup.py

Paste the Script & Save the File

 In Nano:
 Press Ctrl+O to save.
 Press Enter to confirm the filename.
 Press Ctrl+X to exit.

 In Vim:
 Press Esc to enter command mode.
 Type :wq and press Enter to save and exit.

Make the Script Executable 

  nano +x prometheus_setup.py


Run the Script:
python3 prometheus_setup.py (or) sudo python3 prometheus_setup.py
